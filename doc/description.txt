EP Mobile provides cardiology tools for cardiac electrophysiologists, cardiologists and other health care workers who deal with cardiac arrhythmias. This is a work in progress. New modules will be added every few weeks, so be sure and keep the program updated.

Current features: 
- Dofetilide (Tikosyn) dose calculator
- QTc calculator
- Choose QTc formula (Bazett, Fridericia, Sagie or Hodges)
- Cycle length to heart rate calculator
- Atrial fibrillation risk scores (CHADS, CHADS-VASc, HAS-BLED)
- Dabigatran (Pradaxa) dose calculator
- Warfarin (Coumadin) weekly dose calculator
- Wide complex tachycardia algorithms
- NEW: Normal EP values
- NEW: ARVC/D Diagnosis

Categories: Medicine, Cardiology, Electrophysiology, Arrhythmia.
