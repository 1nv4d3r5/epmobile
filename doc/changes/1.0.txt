EP Mobile v1.0 Release Notes

Changes since v0.13

* First full integer version!

* Restructured menu lists.

* Fixed bug in ARVC/D 2010 criteria (echo minor fractional shortening criterion was incorrect).

* Added Notes button to Brugada ECG patterns, providing explanation of figure.

* Added error checking dialog to Estes LVH module.

* Eliminated right facing arrows in lists, per new Android style guidelines.

* Added source code link to About dialog.

* New icons (actually added in v0.13 but not noted in changes list then).
